5.6.13、实体类对象或其集合值复制工具
1、VO与Map之间对值进行复制的工具，复制依据：遍历map中的key或bean中的字段名，仅当map中的键与bean的字段名相等时复制
类：com.edk4j.core.util.ValueCopyBetweenBeanAndMapUtil
方法如下：
public static <T> Map<String, Object> beanToMapViaSpring(T object) 将实体类对象的字段名-字段值以key-value存储到map中
public static <T> T mapToBeanViaJackson(Map<String, Object> map, T object) 将map中的key与vo的字段名一致的value设置为vo的字段值
public static <T> List<Map<String, Object>> listOfBeanConvertToListOfMap(List<T> objList) 将List<T>转换为List<Map<String, Object>>
public static <T> List<T> listOfMapConvertToListOfBean(List<Map<String, Object>> maps, Class<T> clazz) 将List<Map<String,Object>>转换为List<T>
public static Map<String, Object> beanToMapViaJackson(Object vo) 将实体类对象的字段名-字段值以key-value存储到map中
public static <T> T mapToBeanViaJackson(Map<String, Object> kvMap, Class<T> toValueType) 将map中的key与vo的字段名一致的value设置为vo的字段值
2、实体类对象属性copy工具
类：com.edk4j.core.util.PropertiesCopyUtil
用于复制实体类对象的属性到另一个实体类对象，方法清单：
public static <T> T copyProperties(Object sourceObject,Class<T> destClazz) 创建一个新的destClazz类型的对象，然后将sourceObject对象里的属性值，复制到新对象中去，只复制同名属性！
public static void copyProperties(Object sourceObject,Object targetObject) 将sourceObject对象里的属性值，复制到targetObject的同名属性中去，只复制同名属性！
3、实体类对象集合copy工具
类：com.edk4j.core.util.CollectionUtil
一般用在spring+mybatis开发套路中，从数据库查询得到的是数据库映射对象，返回给前台的是返回类型的对象，此时就可以用这个工具类创造返回对象。
目前提供了一个方法：用于将一个列表里的对象的属性值，逐个复制到另一列表的对象中。
方法清单：
public static <T> List<T> transformList(List<?> sourceList, final Class<T> destClazz) 创建一个新的集合，集合中实体类的类型为destClazz，新集合的大小等于sourceList的大小，然后遍历sourceList集合，将实体类的同名属性值复制到新集合中序列号相同的实体类中
