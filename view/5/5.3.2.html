5.3.2、application.properties
edk4j平台中使用 .properties格式的文件，配置spring参数。
文件位置： ${项目路径}/src/main/resources/springConfig/application.properties
本文件中配置的是“公共性质”的属性，对于环境相关的属性（开发环境、测试环境、生产环境等）配置在具体的 application-{profile}.properties中
配置在该文件中的属性必须是spring中的允许属性，其他属情配置到其他文件中。
本文件中配置的内容有“数据库类型、i18n位置定义、mybatis位置定义等等”，详见spring.io中对于springBoot的介绍文档。
下面是一个样例配置： 
security.basic.enabled =true
spring.application.name = my_project_name
#specify the request contextPath
server.context-path = /my_project_name
#server.port = 8080

#if jpa is unused, to config database.name in profile config file
#ORACLE MYSQL DB2
spring.jpa.database = MYSQL
#we use JpaTransactionManager as default
spring.data.jpa.repositories.enabled=true
spring.jpa.open-in-view=false
#we need't default dialect , it can create foreign keys and other dependency
#com.edk4j.core.config.OracleDialectWithoutFK
#com.edk4j.core.config.MysqlDialectWithoutFK
#com.edk4j.core.config.DB2DialectWithoutFK
spring.jpa.properties.hibernate.dialect=com.edk4j.core.config.MysqlDialectWithoutFK
spring.jpa.properties.hibernate.format_sql = true
spring.jpa.properties.hibernate.cache.user_query_cache = false
spring.jpa.properties.hibernate.max_fetch_depth = 2
spring.jpa.properties.hibernate.temp.use_jdbc_metadata_defaults = false
spring.jpa.show-sql = false
spring.jpa.generate-ddl = false
#needless this naming strategy,the name of field in table is designed by developer
#spring.jpa.hibernate.naming-strategy = com.edk4j.core.config.JPATableDDLNamingStrategy
spring.jpa.hibernate.ddl-auto = none

#mybatis
#mybatis.config-location=classpath:mybatis/mybatis-config.xml
#mybatis.mapper-locations=classpath*:mybatis/mapper/**/*.xml

# TRANSACTION (TransactionProperties)
# Default transaction timeout in seconds.
spring.transaction.default-timeout= 30
# Perform the rollback on commit failures.
#spring.transaction.rollback-on-commit-failure= 

#banner
#banner.location = banner.txt
#banner.image.location =
#banner.charset = UTF-8 

#useless in tomcat 1-6, and you shoud set this contextParam in web.xml
spring.profiles.active = development
#spring.config.location = classpath:springConfig/application-product.properties

#实现配置文件自定义，不依赖spring的profile机制以及文件命名application-{xxx}.properties的约束，可以通过在 ApplicationBootConfig.java类中添加注解"@PropertySource" ，样例如下
#@PropertySource(
#	value = {

#		"classpath:edk4j/edk4j.properties",

#	}
#)

#i18n set
spring.messages.basename=springConfig/i18n/messages, springConfig/i18n/error
spring.messages.encoding=UTF-8

spring.jmx.default-domain=outwork
